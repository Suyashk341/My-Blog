# My-Blog

This repository now contains a static MVP website inspired by CoachingDekho-style discovery flows.

## What is included
- Hero section with exam/city filters
- Coaching institute cards with dynamic filtering
- Features section
- Counselling lead form with confirmation message

## Run locally
You can open `index.html` directly, or serve it:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.
